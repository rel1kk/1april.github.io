VPN Import

Импорт напрямую с RAW-ссылки. Без sub.txt, без fallback VLESS, без лишнего мусора.

Готово.
На ПК открой страницу на телефоне.
Как это работает:

1. Кнопка сначала копирует RAW-ссылку в буфер
2. Потом пытается открыть приложение через несколько схем
3. Если приложение тупое — ссылка уже в буфере, вставляешь вручную
