Subscription Import

Страница открывает подписку в приложении или даёт запасной вариант через копирование / QR.

Готово.
Открой эту страницу на телефоне или отсканируй QR.
Если импорт не сработал:

1. Нажми «Скопировать ссылку»
2. Открой приложение вручную
3. Найди Import / Add Subscription / Add from URL
4. Вставь ссылку вручную
